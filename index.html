⚙️ Configure o Supabase para sincronizar entre dispositivos (é grátis!)
  1. Acesse supabase.com → crie uma conta → crie um projeto
  2. No painel vá em SQL Editor e execute este comando:
    create table esmaltes (id uuid primary key default gen_random_uuid(), name text not null, date text, img text, created_at timestamptz default now());
  3. Ainda no SQL Editor, libere acesso público:
    alter table esmaltes enable row level security; create policy "public" on esmaltes for all using (true) with check (true);
  4. (Opcional) Para fotos da galeria: vá em Storage → crie um bucket chamado fotos → marque como público
  5. Vá em Project Settings → API → copie a Project URL e a chave anon public
  6. Cole os dois valores no script abaixo substituindo COLE_URL_AQUI e COLE_ANON_KEY_AQUI
